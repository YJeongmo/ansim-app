# 변경 이력 (Changelog)

## [2025-09-15] GPT-5-mini 성능 최적화

### 🚀 새로운 기능
- **GPT-5-mini 모델 적용**: 비용 95% 이상 절약
- **응답 캐싱 시스템**: 동일 요청에 대해 즉시 응답 (< 1초)
- **자동 재시도 로직**: 네트워크 오류 시 최대 3회 재시도
- **성능 최적화 설정**: 더 상세하고 일관된 분석 결과

### 🔧 개선사항
- **API 타임아웃 최적화**: 연결 30초, 읽기 2분으로 설정
- **JSON 응답 강제**: 파싱 오류 감소
- **사고 과정 활성화**: 분석 품질 향상
- **캐시 관리**: 24시간 TTL, 최대 100개 응답 캐시

### 📊 성능 지표
- **비용 절약**: 95% 이상 (GPT-4 → GPT-5-mini)
- **응답 속도**: 캐시된 요청 < 1초
- **안정성**: 자동 재시도로 네트워크 오류 복구

### 🛠️ 기술적 변경사항
- `OpenAIService.java`: 성능 최적화 설정 추가
- `AnsimYoyangApplication.java`: @EnableRetry 어노테이션 추가
- `build.gradle`: Spring Retry 의존성 추가
- 캐싱 메커니즘: SHA-256 기반 고유 키 생성
