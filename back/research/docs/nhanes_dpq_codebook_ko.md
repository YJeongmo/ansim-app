## NHANES DPQ(PHQ-9) 코드북 요약

### 파일/키
- 파일명: `DPQ_[주기].XPT` (예: `DPQ_L`=2021–2022)
- 키: `SEQN` (응답자 식별자, 다른 모듈과 조인)

### 문항(지난 2주 빈도)
- `DPQ010` 흥미/즐거움 감소
- `DPQ020` 우울/절망감
- `DPQ030` 수면 문제
- `DPQ040` 피로/에너지 부족
- `DPQ050` 식욕 변화
- `DPQ060` 자기 비하/실패감
- `DPQ070` 집중 곤란
- `DPQ080` 느려짐/초조
- `DPQ090` 자살 생각
- `DPQ100` 기능장애(일/집안일/대인관계 영향)

### 값 코딩(일반)
- 0 전혀 없음 / 1 며칠 동안 / 2 일주일 이상 / 3 거의 매일
- 기능장애(`DPQ100`): 0 전혀 어렵지 않다 / 1 약간 / 2 매우 / 3 극히
- 특수코드(미응답/거절/모름 등)는 결측 처리 후 점수 산출 필요

### 점수/중증도
- 총점 = `DPQ010`–`DPQ090` 합(0–27)
- 중증도 예: 0–4 none, 5–9 mild, 10–14 moderate, 15–19 moderately severe, 20–27 severe

### 표본설계/가중치
- 가중치: `WTINT2YR` (면접)
- 층화/PSU: `SDMVSTRA`, `SDMVPSU`
- 인구통계(`DEMO_[주기].XPT`)에서 `SEQN`으로 조인

### 참고
- NHANES 공식: 주기별 DPQ 페이지(코드북/데이터)

## NHANES 다운로드 가이드(무엇을 받으면 되나요)

### 단일 주기(예: 2021–2022)로 시작
1. NHANES 포털 방문: `https://wwwn.cdc.gov/Nchs/Nhanes/`
2. 상단 메뉴에서 Questionnaire Data 선택 → 대상 주기(2021–2022) 진입
3. "Depression Screener (DPQ)" 페이지에서 다음 파일 다운로드:
   - Data File: `DPQ_L.XPT`
   - Documentation(Codebook): PDF/HTML 문서
4. 동일 주기의 인구통계 페이지에서 다음 파일 다운로드:
   - Data File: `DEMO_L.XPT`
   - Documentation(Codebook)
5. 프로젝트 배치 권장 위치: `back/research/data/` (원본 유지)

### 다주기 병합(표본 수 확장)
- 주기별 파일 목록(예시):
  - 2011–2012: `DPQ_G.XPT`, `DEMO_G.XPT`
  - 2013–2014: `DPQ_H.XPT`, `DEMO_H.XPT`
  - 2015–2016: `DPQ_I.XPT`, `DEMO_I.XPT`
  - 2017–2018: `DPQ_J.XPT`, `DEMO_J.XPT`
  - 2019–2020: `DPQ_K.XPT`, `DEMO_K.XPT`
  - 2021–2022: `DPQ_L.XPT`, `DEMO_L.XPT`

병합 시 `SEQN`으로 이어붙이고, 분석 목적에 따라 2년 가중치(`WTINT2YR`) 처리와 주기 더미/재가중 고려를 문서화합니다.
